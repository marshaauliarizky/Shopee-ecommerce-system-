## Features

| Feature | Description |
|---------|-------------|
| Sign Up & Login | User registration via email or phone number with OTP verification |
| Product Search & Filtering | Keyword search with filters for price, rating, and location |
| Shopping Cart & Checkout | Add, update, or remove items and proceed to checkout |
| Payment System | Multiple payment methods with auto-confirmation |
| Order Tracking | Real-time delivery status integrated with logistics services |
| Rating & Review | Submit ratings and reviews after order completion |
